1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Some, Any, No, None
수량 표현 — some과 any의 차이

Some → 긍정문, 제안, 요청에 사용. "약간의, 몇 개의"

Any → 부정문, 의문문에 사용. "어떤, 아무"

No → 명사 앞에서 강한 부정. "아무도 없는"

None → 대명사로 사용. "아무것도 없음"

  • I have some apples.나는 사과가 조금 있어요
  • Do you have any apples?사과 있어?
  • I have no apples.나는 사과가 하나도 없어
  • None of the apples are fresh.사과 중 아무것도 신선하지 않아
2
문법 규칙GRAMMAR RULES
SOME
Affirmative / Offers / Requests
I have some apples.
ANY
Negative / Questions / Conditions
Do you have any apples?
NO
Before a noun (strong negation)
I have no apples.
NONE
Pronoun (stands alone)
None of them are fresh.
  • Some: 긍정문, 제안(Would you like some?), 요청(Can I have some?) 사용
  • Any: 부정문(I don't have any), 의문문(Do you have any?), 조건문(If any) 사용
  • No: 명사 앞에서 강한 부정, 동사는 긍정형 (I have no money)
  • None: 대명사로 독립 사용, none of + 명사 (None of them came)
  • 불가산/가산 명사: some milk, some apples / any milk, any apples 모두 가능

자주 쓰이는 패턴COMMON PATTERNS
긍정문 · AFFIRMATIVE
S + V + some + 명사
I have some books. / There are some students.
부정문 · NEGATIVE
S + don't/doesn't + V + any + 명사
I don't have any money.
의문문 · QUESTION
Do/Does + S + V + any + 명사?
Do you have any questions?
강한 부정 · STRONG NEGATION
S + V + no + 명사 / None of + 명사
I have no time. / None of them came.

흔한 실수COMMON MISTAKES
I don't have some money.
I don't have any money.
부정문에서는 some 대신 any 사용
Do you have some questions?
Do you have any questions?
의문문에서는 some 대신 any 사용
I have no any idea.
I have no idea.
no와 any를 함께 쓰지 않음
None apples are fresh.
None of the apples are fresh.
none은 대명사이므로 none of + the + 명사
She doesn't have no friends.
She has no friends.
이중 부정 금지: doesn't + no 함께 쓰지 않음
10
레벨업 팁LEVEL-UP TIPS
TIP 1 · 발음
Some [sʌm] 약하게, Any [ˈeni] 첫 음절 강세, No [noʊ] 단호하게, None [nʌn] 강하게 발음합니다.
TIP 2 · 뉘앙스 차이
"Some"은 친절한 느낌, "Any"는 중립적, "No/None"은 단호한 느낌을 줍니다.
TIP 3 · 제안에서 Some
정중한 제안에서는 any 대신 some을 사용합니다: "Would you like some tea?"
TIP 4 · 조건문에서 Any
"Any"는 조건문에서도 자주 사용됩니다: If you have any questions, ask me.

연습문제PRACTICE EXERCISES

빈칸에 some, any, no, none 중 알맞은 것을 넣으세요.

  • I have ___ books. (긍정) (some)
  • I don't have ___ books. (부정) (any)
  • Do you have ___ books? (의문) (any)
  • I have ___ books. (강한 부정) (no)
  • ___ of the books are useful. (대명사) (None)
  • Would you like ___ tea? (제안) (some)
  • If you have ___ problems, call me. (조건문) (any)
  • There are ___ seats left. (부정 강조) (no)
