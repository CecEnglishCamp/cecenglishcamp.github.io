1. 기본 개념 (Basic Concept)

CONDAdvanced Conditionals

고급 조건문: unless / provided that
조건을 표현하는 다양한 접속사 사용
unless: ~하지 않으면 (부정 조건)
provided that: ~하는 경우에만 (제한 조건)
규칙, 경고, 제한, 조언 등에서 자주 사용됨

Unless: You won’t pass unless you study.

Provided that: You can join provided that you finish your work.

2. 문법 규칙 (Grammar Rules)

  • Unless = If not (~하지 않으면)
  • Provided that = Only if (~하는 경우에만)
  • 기본 구조: 접속사 + 주어 + 동사, 주절
  • 시제는 현재, 미래, 과거 모두 가능
  • 쉼표는 접속사가 문장 앞에 올 때 사용
  • 조건이 충족되지 않으면 결과가 달라짐
  • 공식 문서, 규칙, 계약서 등에서 자주 등장

3. 긍정문 예시 (Positive Examples)

Unless 예시 (부정 조건)

  • You won’t pass unless you study.
  • We can’t start unless everyone is here.
  • She won’t go unless he invites her.
  • I won’t sleep unless I finish this.
  • They won’t play unless it stops raining.

Provided that 예시 (제한 조건)

  • You can join provided that you finish your work.
  • We’ll go provided that the weather is good.
  • She’ll help provided that you ask nicely.
  • They’ll allow it provided that you follow the rules.
  • I’ll agree provided that we leave early.

5. 상황별 대화 흐름 (Situational Dialogues)

6. 문장 결합 연습 (Sentence Combining)

  • If you don’t study, you won’t pass. → You won’t pass unless you study.
  • If everyone isn’t here, we can’t start. → We can’t start unless everyone is here.
  • If he doesn’t invite her, she won’t go. → She won’t go unless he invites her.
  • If I don’t finish this, I won’t sleep. → I won’t sleep unless I finish this.
  • If it doesn’t stop raining, they won’t play. → They won’t play unless it stops raining.
  • If you finish your work, you can join. → You can join provided that you finish your work.
  • If the weather is good, we’ll go. → We’ll go provided that the weather is good.
  • If you ask nicely, she’ll help. → She’ll help provided that you ask nicely.
  • If you follow the rules, they’ll allow it. → They’ll allow it provided that you follow the rules.
  • If we leave early, I’ll agree. → I’ll agree provided that we leave early.

7. 실생활 사용 예시 (Real-Life Usage)

  • 학교 규칙: You can retake the test provided that you attend all review sessions.
  • 직장 조건: You may leave early provided that your tasks are complete.
  • 가정 상황: You won’t get dessert unless you eat your vegetables.
  • 온라인 설정: The app won’t work unless you update it.
  • 병원 안내: She can go home provided that the doctor approves.
  • 여행 계획: We’ll go hiking provided that the weather is clear.
  • 회의 규칙: The meeting won’t start unless the manager arrives.

8. 자주 쓰이는 패턴 (Common Patterns)

  • Unless + 주어 + 동사, 주절
  • Provided that + 주어 + 동사, 주절
  • You won’t ___ unless you ___
  • You can ___ provided that you ___
  • We’ll ___ provided that ___
  • She won’t ___ unless ___
  • They’ll ___ provided that ___
  • I’ll ___ unless ___
  • It won’t ___ unless ___
  • You may ___ provided that ___

9. 흔한 실수 (Common Mistakes)

  • ❌ You won’t pass unless you won’t study → ✅ You won’t pass unless you study
  • ❌ Provided that you will finish, you can join → ✅ You can join provided that you finish
  • ❌ Unless everyone isn’t here, we can’t start → ✅ We can’t start unless everyone is here
  • ❌ She’ll help provided you ask nicely → ✅ She’ll help provided that you ask nicely
  • ❌ I’ll agree unless we leave early → ✅ I’ll agree provided that we leave early

10. 레벨업 팁 (Level-Up Tips)

  • Unless는 “if not”과 같은 의미지만 더 간결하고 자연스러움
  • Provided that은 “only if”와 같은 의미로 제한 조건을 강조함
  • 접속사가 문장 앞에 올 경우 쉼표 사용 (Unless you study, you won’t pass)
  • 시제는 현재, 미래, 과거 모두 가능
  • 공식 문서, 규칙, 계약서 등에서 provided that 자주 사용됨
  • 구어체에서는 unless가 더 자주 쓰임
  • 문장 결합 연습을 통해 조건 강조 표현에 익숙해지기

11. 연습문제 (Practice Exercises)

객관식

  1. Which is correct?
    A. You won’t pass unless you won’t study.
    B. You won’t pass unless you study.
  2. Choose the correct form:
    ___ you finish your work, you can join.
    A. Provided that
    B. Unless
  3. Fill in: We can’t start ___ everyone is here.
    A. unless
    B. provided that

1. B. 'unless' 자체가 부정의 의미를 포함하므로, 뒤에 won't를 쓰지 않습니다.

2. A. 일정한 조건 하에 허가하는 상황이므로 'provided that'이 적절합니다.

3. A. '모두가 여기에 있지 않으면'이라는 부정 조건을 나타내므로 'unless'가 적절합니다.
