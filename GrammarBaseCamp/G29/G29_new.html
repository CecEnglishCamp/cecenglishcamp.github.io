First Conditional
첫 번째 조건문 — if + will 완전 정복
기본 개념 문법 규칙 긍정문 의문문 대화 흐름 문장 결합 실생활 패턴 흔한 실수 레벨업 팁 연습문제 자가진단
1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
If + Present, Will + Base
현실적인 조건과 결과 — "만약 ~한다면, ~할 것이다"

First Conditional은 현실적으로 일어날 가능성이 있는 조건과 그 결과를 표현합니다. 미래에 실제로 발생할 수 있는 상황을 가정할 때 사용하는 중요한 문법입니다.

If + 현재 시제, Will + 동사원형의 구조로, "만약 ~한다면, ~할 것이다"라는 의미를 가집니다.

2
문법 규칙GRAMMAR RULES
조건절 · IF CLAUSE
If + 현재 시제
If it rains tomorrow,
결과절 · MAIN CLAUSE
주어 + will + 동사원형
we will stay home.
부정문 · NEGATIVE
don't/doesn't / won't
If you don't study, you won't pass.
의문문 · QUESTION
Will + 주어 + 동사원형 + if...?
Will you help me if I ask?
  • If절에는 현재 시제를 사용합니다. (will 사용 불가!)
  • 결과절에는 will + 동사원형을 사용합니다.
  • If절이 먼저 오면 쉼표(,)를 사용합니다.
  • Will절이 먼저 오면 쉼표를 사용하지 않습니다.
  • 3인칭 단수에는 동사에 -s/-es를 붙입니다. (If she arrives...)
3
긍정문 예시POSITIVE EXAMPLES
  • If you study, you will pass the test.공부하면 시험에 합격할 것이다.
  • If we leave now, we'll catch the train.지금 출발하면 기차를 탈 수 있을 것이다.
  • If she calls, I will answer.그녀가 전화하면 받을 것이다.
  • If it rains, we will cancel.비가 오면 취소할 것이다.
  • If she arrives early, we can start.그녀가 일찍 도착하면 시작할 수 있다.
  • If they need help, I will assist.그들이 도움이 필요하면 도울 것이다.
  • If you hurry, you will catch the bus.서두르면 버스를 탈 수 있을 것이다.
  • If I see him, I'll tell him.그를 보면 말해줄 것이다.
  • If we save money, we will buy a car.돈을 모으면 차를 살 것이다.
  • We will stay home if it rains.비가 오면 집에 있을 것이다. (순서 바꾸기)

상황별 대화 흐름SITUATIONAL DIALOGUES
📅 약속 잡을 때 · MAKING PLANS
A: If you're free tomorrow, we'll meet for coffee.
B: Sounds good. If it rains, we can go indoors.
🎬 계획 세울 때 · PLANNING
A: If we finish early, we'll watch a movie.
B: Okay. If the movie is long, we'll order pizza.
💡 조언할 때 · GIVING ADVICE
A: If you practice every day, you'll improve quickly.
B: Thanks. If I need help, I'll ask you.
🏔️ 주말 계획 · WEEKEND PLANS
Tom: What are your plans for the weekend?
Lisa: If the weather is nice, I'll go hiking. What about you?
Tom: If I finish my work, I'll join you.
Lisa: Great! If you come, we'll have lunch together.
Tom: Perfect. If it rains, we can go to a museum instead.

문장 결합 연습SENTENCE COMBINING
It rains. We stay home.
→ If it rains, we will stay home.
You study hard. You pass the exam.
→ If you study hard, you will pass the exam.
She calls me. I answer.
→ If she calls me, I will answer.
You don't hurry. You miss the train.
→ If you don't hurry, you will miss the train.
He invites me. I don't go.
→ If he invites me, I won't go.
It doesn't rain. We go out.
→ If it doesn't rain, we 'll go out.

실생활 사용 예시REAL-LIFE USAGE
  • If you don't study, you won't pass.📖 학교 — 공부하지 않으면 합격하지 못할 것이다.
  • If it doesn't rain, we'll go out.🌤️ 날씨 — 비가 오지 않으면 나갈 것이다.
  • If he invites me, I won't go.🎉 사교 — 그가 초대해도 가지 않을 것이다.
  • If I have time, I will help you.💼 직장 — 시간이 있으면 도와줄게.
  • If you eat too much, you'll feel sick.🍽️ 건강 — 너무 많이 먹으면 아플 거야.
  • If we leave early, we'll avoid traffic.🚗 교통 — 일찍 출발하면 교통체증을 피할 수 있다.
  • If you call me, I'll pick you up.📱 일상 — 전화하면 데리러 갈게.

자주 쓰이는 패턴COMMON PATTERNS
IF절 먼저 · IF CLAUSE FIRST
If + 현재 시제, 주어 + will + 동사원형
If it rains, we will stay home.
WILL절 먼저 · MAIN CLAUSE FIRST
주어 + will + 동사원형 + if + 현재 시제
We will stay home if it rains.
부정 조건 · NEGATIVE CONDITION
If + 주어 + don't/doesn't + 동사원형, 주어 + will/won't...
If you don't study, you won't pass.
의문 조건 · CONDITIONAL QUESTION
Will + 주어 + 동사원형 + if + 현재 시제?
Will you help me if I ask?
9
흔한 실수COMMON MISTAKES
✗ If it will rain, we will stay home.
✓ If it rains, we will stay home.
If절에는 현재 시제를 사용합니다. Will을 사용하지 않습니다.
✗ If you will come, I will be happy.
✓ If you come, I will be happy.
If절에서는 will을 생략하고 현재 시제만 사용합니다.
✗ If she studies, she pass the exam.
✓ If she studies, she will pass the exam.
결과절에는 will을 사용해야 합니다.
✗ If you don't hurry, you miss the train.
✓ If you don't hurry, you will miss the train.
결과절에 will을 빠뜨리지 않도록 주의합니다.
10
레벨업 팁LEVEL-UP TIPS
TIP 1 · If절 = 현재 시제
If절에는 절대 will을 쓰지 않습니다! 미래를 말하더라도 현재 시제를 사용하세요.
TIP 2 · 축약형 연습
회화에서는 will을 'll로 줄여 말합니다. "I'll help you if you need."
TIP 3 · 순서 바꾸기
If절과 Will절의 순서를 바꿀 수 있습니다. If절이 먼저 오면 쉼표, 뒤에 오면 쉼표 없이!
TIP 4 · 실생활 연습
First Conditional은 계획, 약속, 조언 등 일상 대화에서 매우 자주 사용됩니다. 많이 말해보세요!

연습문제PRACTICE EXERCISES

빈칸에 알맞은 동사 형태를 넣으세요.

  • If it _____ (be) sunny, we _____ (go) to the park. (is / will go)
  • If you _____ (call) me, I _____ (answer). (call / will answer)
  • If she _____ (study), she _____ (pass) the exam. (studies / will pass)
  • If I _____ (have) time, I _____ (help) you. (have / will help)
  • If they _____ (not hurry), they _____ (miss) the bus. (don't hurry / will miss)
  • _____ you _____ (come) if I _____ (invite) you? (Will / come / invite)
  • If it _____ (rain), we _____ (not go) out. (rains / won't go)
  • If we _____ (leave) now, we _____ (catch) the train. (leave / will catch)

자가진단 체크리스트SELF-CHECK
  • First Conditional의 기본 구조를 설명할 수 있다.
  • If절에는 현재 시제를 사용한다는 것을 안다.
  • Will절에는 will + 동사원형을 사용한다는 것을 안다.
  • If절과 Will절의 순서를 바꿀 수 있다.
  • 부정문을 만들 수 있다 (don't/doesn't, won't).
  • 의문문을 만들 수 있다 (Will + 주어 + 동사원형).
  • 일상 대화에서 First Conditional을 자연스럽게 사용할 수 있다.
