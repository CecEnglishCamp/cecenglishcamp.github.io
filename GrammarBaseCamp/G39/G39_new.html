Still, Yet, Already
아직 / 이미 / 벌써
기본 개념문법 규칙긍정문의문문대화 흐름문장 결합실생활패턴흔한 실수레벨업 팁연습문제자가진단
1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Still, Yet, Already
아직 계속 / 아직 안 / 이미 벌써

Still → 어떤 일이 계속되고 있음. 긍정문에서 동사 앞에 위치. I'm still working.

Yet → 어떤 일이 아직 발생하지 않음. 의문문/부정문에서 문장 끝에 위치. I haven't eaten yet.

Already → 어떤 일이 이미 발생함. 긍정문에서 현재완료와 결합. I've already finished.

  • I'm still working.아직 일하고 있어요 (계속 진행 중)
  • I haven't eaten yet.아직 먹지 않았어요 (미완료)
  • I've already finished.이미 끝냈어요 (완료됨)
  • Have you finished yet?끝났나요? (확인)
2
문법 규칙GRAMMAR RULES
STILL · 계속
동사 앞 / be동사 뒤
I still live here. / I'm still waiting.
YET · 아직
문장 끝 (부정/의문)
I haven't finished yet.
ALREADY · 이미
have와 과거분사 사이
I've already eaten.
비교 · COMPARISON
세 단어의 시제 차이
still=진행 / yet=미완료 / already=완료
  • Still: 일반적으로 긍정문에서 사용, 동사 앞에 위치 (She still lives here)
  • Yet: 의문문/부정문에서 사용, 문장 끝에 위치 (Have you eaten yet?)
  • Already: 긍정문에서 사용, 현재완료 시제와 자주 결합 (I've already eaten)
  • Still은 현재진행/현재시제와 자주 사용됨
  • Yet은 현재완료/미완료 동작 확인에 사용됨
  • Already는 놀람/예상보다 빠른 완료를 표현함

상황별 대화 흐름SITUATIONAL DIALOGUES
🏫 학교 상황 · SCHOOL
A: Have you finished your homework yet?
B: No, I'm still working on it.
A: I already finished mine.
B: Lucky you!
✈️ 여행 상황 · TRAVEL
A: Did you book the hotel yet?
B: Yes, I already did it.
A: Are you still planning the schedule?
B: Yes, I need more time.
💼 직장 상황 · WORK
A: Have they submitted the report yet?
B: No, they're still editing.
A: I already sent mine.
B: You're always early!
💻 온라인 상황 · ONLINE
A: Have you logged in yet?
B: Yes, I already joined the meeting.
A: Is Sarah still offline?
B: Yes, she hasn't joined yet.
🏠 가족 상황 · FAMILY
A: Did Mom cook dinner yet?
B: No, she's still preparing.
A: I already set the table.
B: Great, thanks!

문장 결합 연습SENTENCE COMBINING

→ I'm still working.
I finished my homework. It's done.
→ I've already finished my homework.
I haven't eaten. Not done yet.
→ I haven't eaten yet.
She went home. Before expected.
→ She's already gone home.
He's studying. Not stopped.
→ He's still studying.
They haven't arrived. Waiting.
→ They haven't arrived yet.
I told you. Before now.
→ I already told you.
It's raining. Continues.
→ It's still raining.
She bought tickets. Done early.
→ She already bought the tickets.

10
레벨업 팁LEVEL-UP TIPS
TIP 1 · 위치가 핵심
Still은 동사 앞/be동사 뒤, Yet은 문장 끝, Already는 have와 과거분사 사이!
TIP 2 · 문장 유형
Still은 주로 긍정문, Yet은 부정문/의문문, Already는 긍정문에서 사용합니다.
TIP 3 · Still + 부정 = 강조
"He still hasn't called."는 "아직도 전화 안 했어"로 불만이나 놀라움을 강조합니다.
TIP 4 · Already + 의문문 = 놀람
"Have you already finished?"는 "벌써 끝났어?"로 예상보다 빠른 완료에 놀라움을 표현합니다.
