So vs Such
너무 ~한 — so와 such의 차이
기본 개념문법 규칙긍정문의문문대화 흐름문장 결합실생활패턴흔한 실수레벨업 팁연습문제자가진단
1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
So vs Such
형용사/부사 강조(so) vs 명사구 강조(such)

So → 형용사나 부사를 강조: She is so tired. / He runs so fast. / I have so much work.

Such → 명사구를 강조: It was such a good idea. / They are such nice people.

  • She is so kind.그녀는 너무 친절해요 (so + 형용사)
  • It was such a great party.정말 멋진 파티였어요 (such + a + 명사)
  • He runs so fast.그는 너무 빨리 달려요 (so + 부사)
  • She is such a good teacher.그녀는 정말 좋은 선생님이에요
2
문법 규칙GRAMMAR RULES
SO + 형용사/부사
so + adj/adv
so fast, so beautiful, so well
SUCH + 명사구
such + (a/an) + adj + noun
such a nice day
SO...THAT · 결과
so + adj + that + 결과
so tired that she fell asleep
SUCH...THAT · 결과
such + 명사구 + that + 결과
such a good idea that...
  • So는 단독 형용사/부사 강조, Such는 명사 포함 강조
  • So + much/many: I have so much work. / There are so many people.
  • Such + a/an + 형용사 + 단수명사: It was such a long day.
  • Such + 형용사 + 복수명사: They are such nice people. (a/an 없음)
  • Such + 추상명사: We had such fun. (a/an 없음)
  • So/Such...that: 결과를 나타내는 that절과 함께 사용 가능

상황별 대화 흐름SITUATIONAL DIALOGUES
🏫 학교 상황 · SCHOOL
A: The test was so hard!
B: Yes, and it was such a long test too.
A: I studied so much last night.
B: It was such a stressful day.
✈️ 여행 상황 · TRAVEL
A: The beach was so beautiful.
B: It was such a perfect place to relax.
A: We had such a great time.
B: The sunset was so amazing.
💼 직장 상황 · WORK
A: He gave such a clear presentation.
B: Yes, he spoke so confidently.
A: It was such an important meeting.
B: Everyone was so focused.
🏠 가족 상황 · FAMILY
A: Dinner was so delicious.
B: Mom made such a tasty meal.
A: I'm so full now.
B: It was such a nice evening.
🏥 병원 상황 · HOSPITAL
A: The doctor was so kind.
B: She gave such good advice.
A: I feel so much better now.
B: It was such a helpful visit.

문장 결합 연습SENTENCE COMBINING
She was tired. She fell asleep.
→ She was so tired that she fell asleep.
It was a good idea. Everyone agreed.
→ It was such a good idea that everyone agreed.
The weather was cold. We stayed inside.
→ The weather was so cold that we stayed inside.
He gave a long speech. People got bored.
→ He gave such a long speech that people got bored.
The cake was delicious. We ate it all.
→ The cake was so delicious that we ate it all.
She is a good teacher. Students love her.
→ She is such a good teacher that students love her.
They worked hard. They finished early.
→ They worked so hard that they finished early.
It was a beautiful day. We went hiking.
→ It was such a beautiful day that we went hiking.
The noise was loud. We woke up.
→ It was such a loud noise that we woke up.

실생활 사용 예시REAL-LIFE USAGE
  • This app is so easy to use. It's such a helpful tool.💻 온라인 — 이 앱은 너무 사용하기 쉬워요. 정말 유용한 도구예요.
  • The presentation was so clear. He is such a talented speaker.💼 직장 — 발표가 너무 명확했어요. 정말 재능 있는 발표자예요.
  • I'm so excited about the trip. It's going to be such a fun adventure.✈️ 여행 — 여행이 너무 기대돼요. 정말 재미있는 모험이 될 거예요.
  • The weather was so hot that we stayed inside. It was such a humid day.🌤️ 날씨 — 너무 더워서 실내에 있었어요.
  • She works so hard. She's such a dedicated employee.💼 업무 보고 — 너무 열심히 일해요. 정말 헌신적인 직원이에요.

자주 쓰이는 패턴COMMON PATTERNS
SO + 형용사/부사
so tired / so fast / so well
She is so kind. He runs so fast.
SUCH + A/AN + 형용사 + 명사
such a nice day / such an interesting book
It was such a great party.
SO + ADJ + THAT + 결과
so tired that she fell asleep
It was so hot that we stayed inside.
SUCH + 명사구 + THAT + 결과
such a good idea that everyone agreed
He gave such a long speech that...
9
흔한 실수COMMON MISTAKES
✗ She is such tired.
✓ She is so tired.
형용사만 있을 때는 so 사용 (such는 명사구와 함께)
✗ It was so a good idea.
✓ It was such a good idea.
명사가 포함된 구에는 such 사용
✗ I have such much work.
✓ I have so much work.
much/many/few/little 앞에는 so 사용
✗ They are so nice people.
✓ They are such nice people.
형용사 + 명사 구조에는 such 사용
10
레벨업 팁LEVEL-UP TIPS
TIP 1 · 핵심 구분
So + 형용사/부사 (명사 없이), Such + (a/an) + 형용사 + 명사 (명사 포함)
TIP 2 · much/many 주의
so much / so many / so few / so little — 수량 표현 앞에는 항상 so!
TIP 3 · that절 활용
so...that / such...that 구문으로 원인과 결과를 연결할 수 있습니다.
TIP 4 · 감탄 표현
일상에서 감탄할 때 자주 사용합니다. "So good!" / "Such a great idea!"
