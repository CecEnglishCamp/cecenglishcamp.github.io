기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Used To vs Be Used To
과거 습관 표현 vs 현재 익숙함 표현

Used to → 과거의 습관이나 상태를 표현합니다. 지금은 더 이상 그렇지 않다는 의미를 포함합니다. used to + 동사원형

Be used to → 어떤 것에 익숙해졌다는 의미입니다. 현재에도 그 익숙함이 유지됩니다. be used to + 명사/동명사

  • I used to live in Seoul.과거에 서울에 살았지만 지금은 아님
  • I am used to living in Seoul.서울에서 사는 것이 익숙함
  • She used to eat meat.과거엔 고기를 먹었지만 지금은 채식주의자
  • She is used to spicy food.매운 음식에 익숙함
2
문법 규칙GRAMMAR RULES
USED TO · 긍정문
subject + used to + base verb
I used to play soccer.
BE USED TO · 긍정문
subject + be + used to + noun/-ing
I am used to cold weather.
부정문 · NEGATIVE
didn't use to / be not used to
I didn't use to like coffee.
의문문 · QUESTION
Did + subject + use to?
Did you use to live here?
  • Used to: 과거 → 현재 변화를 나타냄 (동사원형과 함께)
  • Be used to: 현재 상태를 나타냄 (명사나 동명사-ing와 함께)
  • 의문문: Did you use to...? (used가 아닌 use)
  • Be used to 시제: I'm / I was / I'll be used to (현재, 과거, 미래 모두 가능)
  • Getting used to: 적응 과정을 표현 (I'm getting used to it.)

  • Is she used to spicy food?매운 음식에 익숙한가요? — Not yet, but she's trying.

상황별 대화 흐름SITUATIONAL DIALOGUES
🏫 학교 상황 · SCHOOL
A: Did you use to study here?
B: Yes, I used to take math classes.
A: Are you used to online learning now?
B: Yes, it's comfortable for me.
💼 직장 상황 · WORK
A: Did he use to work in sales?
B: Yes, for five years.
A: Is he used to remote work now?
B: Yes, he enjoys the flexibility.
🏠 이주 상황 · MOVING
A: Did you use to live in a small town?
B: Yes, my whole life.
A: Are you used to the big city yet?
B: I'm getting used to it slowly.
🍽️ 식단 변화 · DIET
A: Did you use to eat a lot of meat?
B: Yes, every meal.
A: Are you used to being vegetarian now?
B: Yes, I don't miss meat anymore.
💻 기술 적응 · TECHNOLOGY
A: Did your grandpa use to write letters?
B: Yes, he never used email.
A: Is he used to texting now?
B: He's getting used to it.

  • I used to be shy in presentations, but now I'm used to speaking in front of large groups.💼 면접 답변 — 발표에서 수줍었지만 지금은 익숙해요.
  • I used to eat only Western food, but now I'm used to Korean cuisine.🏠 이주 경험 — 서양 음식만 먹었지만 지금은 한식에 익숙해요.
  • He used to handle customer service. He's getting used to leading training sessions.📊 업무 보고 — 고객 서비스를 담당했었고 교육 세션을 이끄는 데 적응 중이에요.
  • The patient used to take pills three times daily. Now he's used to injections once a week.🏥 건강 상담 — 하루 세 번 약을 먹었지만 지금은 주 1회 주사에 익숙해요.

자주 쓰이는 패턴COMMON PATTERNS
USED TO + BASE VERB
used to + 동사원형 (과거 습관)
I used to play soccer every day.
DIDN'T USE TO + BASE VERB
didn't use to + 동사원형 (과거 부정)
I didn't use to like coffee.
BE USED TO + NOUN / VERB-ING
be used to + 명사/동명사 (현재 익숙함)
She's used to waking up early.
GETTING USED TO
getting used to + 명사/동명사 (적응 과정)
I'm getting used to the new schedule.
9
흔한 실수COMMON MISTAKES
✗ I use to play soccer.
✓ I used to play soccer.
과거 습관을 나타낼 때 반드시 used to (과거형)
✗ Did you used to live here?
✓ Did you use to live here?
Did 뒤에는 use to (원형)
✗ I am used to eat spicy food.
✓ I am used to eating spicy food.
be used to 뒤에는 동명사(-ing) 또는 명사
✗ She used to working late.
✓ She is used to working late.
익숙함을 표현할 때는 be used to 사용
10
레벨업 팁LEVEL-UP TIPS
TIP 1 · 핵심 구분
Used to는 과거 습관/상태 → 동사원형과 함께, Be used to는 익숙함 → 명사 또는 동사-ing와 함께!
TIP 2 · 의문문 주의
"Did you use to..."에서 used가 아닌 use를 사용해야 합니다.
TIP 3 · 시제 활용
"Be used to"는 현재, 과거, 미래 시제 모두 가능: I'm / I was / I'll be used to
TIP 4 · 적응 과정 표현
"Getting used to"로 적응 과정을 표현할 수 있습니다. I'm getting used to the new job.
