기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Imperative Commands
명령문 — 지시와 요청 표현

명령문: 상대방에게 지시, 요청, 안내를 할 때 사용하는 문장입니다.

핵심 형태: 문장이 동사 원형으로 시작하며, 주어 (You)는 생략됩니다.

사용 목적: 상대방이 특정 행동을 하도록 요구합니다.

  • Open the door.문을 열어라.
  • Be quiet.조용히 해.
  • Don't touch that.저것 만지지 마.
2
문법 규칙GRAMMAR RULES
POSITIVE COMMAND
Open the door.
BE-VERB COMMAND
Be quiet.
NEGATIVE COMMAND
Don't run.
POLITE REQUEST
Please sit down.
  • 형태: 항상 동사 원형으로 시작 (주어 'You'는 생략)
  • 긍정 명령: 동사 원형 + 목적어/보어 (예: Sit down. / Read this.)
  • 부정 명령: Don't + 동사 원형 (예: Don't run. / Don't talk.)
  • Be동사 명령: Be + 형용사/명사 (예: Be careful. / Be quiet.)
  • 공손한 표현: Please를 문장 맨 앞이나 맨 뒤에 추가 (예: Please open the door. / Wait here, please.)

10
레벨업 팁LEVEL-UP TIPS
TIP 1 · Let's 제안
Let's + 동사 원형은 "우리 ~하자"라는 청유문으로, 명령문과 함께 자주 사용됩니다. (e.g., Let's go to the park, and come back at 5.)
TIP 2 · Don't Be
Be동사의 부정 명령은 Don't be입니다. (e.g., Don't be shy. / Don't be late.)
TIP 3 · 부사 추가
명령문의 동사 뒤에 부사를 추가하여 디테일을 더하세요. (e.g., Walk quickly. / Speak slowly.)
TIP 4 · 강조
명령의 긴급함을 표현할 때는 Don't와 Be에 강세를 줍니다. (e.g., Don't wait!)
11
연습문제PRACTICE EXERCISES

올바른 명령문을 선택하거나 만들어 보세요.

  • 창문을 열어. → Open the window. (주어 생략, 동사 원형으로 시작)
  • 저것을 만지지 마세요. → Don't touch that. (Don't + 동사 원형)
  • ___ careful. → Be careful. (Be동사 명령문)
  • You don't touch that. → Don't touch that. (주어 생략)
  • Be quiet you. → Be quiet. (주어 생략)
  • 문을 닫아 주세요. → Please close the door. (공손한 요청)
  • 조용히 하세요. → Be quiet. (Be동사 명령)
  • 늦지 마. → Don't be late. (Don't be 사용)
