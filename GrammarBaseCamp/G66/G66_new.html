기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Advanced Passive Voice
수동태 — 주어가 동작을 받는 문장 구조 (be + 과거분사)

능동태: 주어가 행동함 → 수동태: 주어가 행동을 당함

기본 구조: be + 과거분사(p.p.) — 행위자보다 결과나 상태에 초점을 맞춤

  • They cleaned the room. → The room was cleaned.능동 → 수동
  • She will write a report. → A report will be written.능동 → 수동
2
문법 규칙GRAMMAR RULES
현재 · PRESENT
is/are + p.p.
The book is read.
과거 · PAST
was/were + p.p.
The book was read.
완료 · PERFECT
has/had been + p.p.
The book has been read.
미래/조동사 · FUTURE
will/can/must be + p.p.
The book will be read.
  • 진행형 수동태: is/was being + p.p. → The book is being read.
  • 행위자(by + 사람): 생략 가능, 강조할 때만 사용
  • 시제에 따라 be동사 형태가 달라짐

💻 온라인 상황 · ONLINE

🏠 가족 상황 · FAMILY

  • The medicine must be taken before meals.🏥 병원 — 약은 식전에 복용해야 한다.
  • This product is made in Korea.🛒 쇼핑 — 이 제품은 한국에서 만들어진다.
  • The flight has been delayed by weather.✈️ 공항 — 날씨로 인해 비행기가 지연되었다.

흔한 실수COMMON MISTAKES
✗ The room cleaned.
✓ The room was cleaned.
be동사 없이 p.p.만 쓸 수 없음
✗ The report is write.
✓ The report is written.
원형이 아닌 과거분사(p.p.) 사용
✗ The homework has been check.
✓ The homework has been checked.
has been 뒤에 p.p. 형태 필수
✗ The email will sent.
✓ The email will be sent.
will 뒤에 be 필수
✗ The task should complete.
✓ The task should be completed.
조동사 수동태: should be + p.p.
✗ The book is reading.
✓ The book is being read.
진행형 수동태: is being + p.p.
✗ The work will finished.
✓ The work will be finished.
will be + p.p. 구조 필수
✗ The letter was wrote.
✓ The letter was written.
write의 과거분사는 written (wrote는 과거형)
10
레벨업 팁LEVEL-UP TIPS
TIP 1 · be동사 핵심
수동태의 핵심은 be + p.p.입니다. 시제에 따라 be동사 형태만 바꾸면 됩니다.
TIP 2 · 과거분사 암기
불규칙 동사의 과거분사를 정확히 기억하세요. write → written, see → seen, take → taken
TIP 3 · 공식 문서에 유용
수동태는 객관적이고 정중한 표현에 유용합니다. 비즈니스 영어, 뉴스, 학술 글에서 자주 사용됩니다.
TIP 4 · 결과에 집중
행위자를 모르거나 결과에 집중할 때 수동태가 효과적입니다. "The window was broken."

자가진단 체크리스트SELF-CHECK
  • 수동태는 be + p.p. 구조가 핵심이라는 것을 안다.
  • 시제에 따라 be동사 형태가 달라진다는 것을 안다.
  • 진행형 수동태(is/was being + p.p.)를 만들 수 있다.
  • 완료형 수동태(has/had been + p.p.)를 만들 수 있다.
  • 조동사 수동태(can/must/should be + p.p.)를 만들 수 있다.
  • 불규칙 동사의 과거분사를 정확히 기억한다.
  • 흔한 실수 8가지를 이해했다.
