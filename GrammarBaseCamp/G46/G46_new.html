Second Conditional
가정법 — If + 과거, would + 동사원형
기본 개념 문법 규칙 긍정문 의문문 대화 흐름 문장 결합 실생활 패턴 흔한 실수 레벨업 팁 연습문제 자가진단
1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Second Conditional
현재 사실과 다른 상상, 희망, 조언을 표현하는 가정법

현실 (Reality): I don't have much money. / She isn't here.

가정 (If-situation): If I had money, I would travel. / If she were here, she would help.

  • If I had wings, I would fly.날개가 있다면 날 텐데. (상상)
  • If I had more time, I would study more.시간이 더 있다면 더 공부할 텐데. (희망)
  • If I were you, I would talk to him.내가 너라면 그와 이야기할 텐데. (조언)
2
문법 규칙GRAMMAR RULES
기본 구조 · STRUCTURE
If + 과거, would + 동사원형
If I had money, I would travel.
순서 변경 · REVERSED
Would + 동사원형 + if + 과거
I would travel if I had money.
부정형 · NEGATIVE
If I didn't..., I would...
If I didn't have homework...
의문형 · QUESTION
What would you do if...?
What would you do if you won?
  • If + 과거, Would + 동사원형 — 기본 구조
  • "If"절에는 과거형 사용 (be동사는 "were" 권장)
  • "Would"는 결과절에서 사용
  • 순서 바꾸기 가능: Would + 주어 + 동사원형 + if + 과거
  • "Could"나 "Might"로 결과절 표현 가능: I might go / I could help
  • 모든 주어에 "were" 사용 (formal): If I were rich... / If he were here...
  • "If I were you..."는 조언할 때 자주 사용하는 표현

긍정문 예시POSITIVE EXAMPLES
  • If I had more money, I would travel the world.돈이 더 있다면 세계 여행을 할 텐데.
  • If she were here, she would help us.그녀가 여기 있다면 도와줄 텐데.
  • If we lived near the beach, we would swim every day.해변 근처에 살면 매일 수영할 텐데.
  • If he studied harder, he would pass the test.더 열심히 공부하면 시험에 붙을 텐데.
  • If I knew the answer, I would tell you.답을 알면 알려줄 텐데.
  • If they invited me, I would go.초대하면 갈 텐데.
  • If I were taller, I would play basketball.키가 더 크면 농구를 할 텐데.
  • If we had a car, we would drive to the mountains.차가 있으면 산으로 드라이브할 텐데.
  • If it didn't rain, we would play outside.비가 안 오면 밖에서 놀 텐데.
  • If I had more time, I would read more books.시간이 더 있으면 책을 더 읽을 텐데.

상황별 대화 흐름SITUATIONAL DIALOGUES

B: That would be great!

문장 결합 연습SENTENCE COMBINING
I don't have money. I can't travel.
→ If I had money, I would travel.
She isn't here. She can't help.
→ If she were here, she would help.
He doesn't study hard. He won't pass.
→ If he studied harder, he would pass.
I don't know the answer. I can't tell you.
→ If I knew the answer, I would tell you.
We don't have a car. We can't drive.
→ If we had a car, we would drive to the mountains.
It rains a lot. We can't play outside.
→ If it didn't rain, we would play outside.
I'm not tall. I can't play basketball.
→ If I were taller, I would play basketball.

자주 쓰이는 패턴COMMON PATTERNS
기본 패턴 · BASIC
If I had + 명사, I would + 동사원형
If I had money, I would travel.
순서 변경 · REVERSED
I would + 동사원형 + if I had + 명사
I would travel if I had money.
조언 패턴 · ADVICE
If I were you, I would + 동사원형
If I were you, I would talk to him.
Wh-의문문 · WH-QUESTION
What would you do if + 과거?
What would you do if you won the lottery?
9
흔한 실수COMMON MISTAKES
✗ If I would have money, I would travel.
✓ If I had money, I would travel.
If절에는 would 사용 불가 — 과거형만 사용
✗ If I was you, I would go.
✓ If I were you, I would go.
가정법에서 be동사는 모든 주어에 "were" 사용 (formal)
✗ If I have money, I would travel.
✓ If I had money, I would travel.
Second conditional은 If + 과거형 (현재형 아님)
✗ If he studied harder, he will pass.
✓ If he studied harder, he would pass.
결과절에는 will 아닌 would 사용
10
레벨업 팁LEVEL-UP TIPS
TIP 1 · 현실과 반대 상황 상상
지금 사실과 반대인 상황을 상상할 때 Second Conditional을 사용하세요. "If I were..." / "If I had..."
TIP 2 · If I were you 활용
"If I were you, I would..." 패턴으로 조언을 주는 연습을 해보세요. 매우 자주 쓰이는 표현입니다.
TIP 3 · could/might도 가능
결과절에 would 대신 could(가능) / might(불확실)도 사용 가능: "If I had time, I could/might help."
TIP 4 · 대화 주제로 활용
"What would you do if...?" 질문으로 대화를 시작해보세요. 영어 말하기 연습에 최고!

연습문제PRACTICE EXERCISES

빈칸에 알맞은 형태를 넣으세요.

  • If I ___ (have) money, I would travel. (had)
  • If she ___ (be) here, she would help. (were)
  • What ___ you ___ (do) if you won? (would ... do)
  • If he ___ (study) harder, he would pass. (studied)
  • I ___ (go) if they invited me. (would go)
  • If I ___ (be) you, I would ask him. (were)
  • ___ you ___ (help) me if I asked? (Would ... help)
  • If it ___ (not rain), we would play outside. (didn't rain)

자가진단 체크리스트SELF-CHECK
  • If + 과거, would + 동사원형 구조를 안다.
  • be동사는 모든 주어에 were를 사용한다는 것을 안다.
  • If I were you 패턴으로 조언할 수 있다.
  • What would you do if...? 질문을 만들 수 있다.
  • If절에 would를 쓰면 안 된다는 것을 안다.
  • 흔한 실수 4가지를 이해했다.
