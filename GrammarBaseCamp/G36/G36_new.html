1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Each vs Every
Each는 개별성 강조, Every는 전체성 강조

Each → 하나하나 따로따로, 개인적/개별적 관점. 2개 이상일 때 사용.

Every → 모든 것을 일반화, 집단적/전체적 관점. 3개 이상일 때 사용.

  • Each student has a locker.학생마다 개별적으로 사물함이 있음
  • Every student studies hard.전체 학생이 일반적으로 열심히 공부함
  • Each person chose a different gift.각자 다른 선물을 선택 (개별성)
  • Every person needs love.모든 사람이 사랑이 필요 (보편성)
2
문법 규칙GRAMMAR RULES
EACH · 개별
Each + 단수 명사 + 단수 동사
Each student is ready.
EVERY · 전체
Every + 단수 명사 + 단수 동사
Every student studies hard.
EACH OF · 부분
Each of + the + 복수 명사
Each of the students is ready.
EVERY + 시간 · TIME
Every + 시간 표현
every day / every week
  • 공통: 둘 다 단수 명사 + 단수 동사와 함께 사용
  • Each of + 복수 명사: 단수 동사와 함께 (Each of the students is ready)
  • Each: 문장 끝에도 위치 가능 (The students each received a book)
  • Every: 시간 표현과 자주 결합 (every day, every week)
  • "Every of"는 불가: Each of는 가능하지만 Every of는 사용 안 함

자주 쓰이는 패턴COMMON PATTERNS
EACH + 단수 명사
Each student has a book. / Each person chose differently.
EACH OF + THE + 복수 명사
Each of the students is ready. / Each of our products is tested.
EVERY + 시간 표현
Every day / Every week / Every other day (하루 걸러)
EVERY SINGLE + 명사 (강조)
Every single detail matters. (모든 세부사항이 중요합니다 - 강조)

10
레벨업 팁LEVEL-UP TIPS
TIP 1 · 핵심 차이
Each는 개별 항목 강조, Every는 전체 집단을 일반화합니다.
TIP 2 · 동사 일치
둘 다 단수 명사 + 단수 동사와 함께 사용됩니다. Each of + 복수명사도 단수 동사!
TIP 3 · 수량 차이
Each는 2개 이상, Every는 3개 이상일 때 일반적으로 사용합니다.
TIP 4 · 강조 표현
"Every single"은 강조 표현으로 사용됩니다. Every single detail matters!

연습문제PRACTICE EXERCISES

알맞은 답을 선택하세요.

  • Each students has a book. / Each student has a book. (Each student has a book.)
  • Every children like candy. / Every child likes candy. (Every child likes candy.)
  • ___ of the apples is fresh. (Each / Every) (Each)
  • 개별성을 강조하는 것은? (Every student / Each student) (Each student)
  • Each of the chairs ___ broken. (is / are) (is)
  • All students have individual lockers. → ___ (Each student has an individual locker.)
  • All apples in the box are sweet. → ___ (Every apple in the box is sweet.)
  • All members contribute equally. → ___ (Every member contributes equally.)

자가진단 체크리스트SELF-CHECK
  • each와 every의 차이를 설명할 수 있다.
  • each는 개별, every는 전체를 강조함을 안다.
  • each of + 복수명사 구조를 쓸 수 있다.
  • every + 단수명사 구조를 쓸 수 있다.
  • 실생활에서 올바르게 구분해 쓸 수 있다.
