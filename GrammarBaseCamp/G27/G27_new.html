Like vs Would Like
좋아하다 vs ~하고 싶다
기본 개념 문법 규칙 긍정문 의문문 대화 문장 결합 실생활 패턴 흔한 실수 레벨업 팁 연습문제 자가진단
1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Like vs Would Like
Like = 일반적인 취향/습관, Would like = 공손한 요청/현재의 바람

Like → 일반적인 취향이나 습관을 표현합니다. 반복적이고 지속적인 선호를 나타냅니다.

Would like → 공손한 요청이나 현재 순간의 희망을 표현합니다. 미래 지향적이며 일시적인 바람을 나타냅니다.

  • I like coffee.나는 커피를 좋아해요. (일반적인 취향)
  • I would like some coffee.커피 좀 주세요. (현재 요청)
  • She likes traveling.여행을 좋아해요. (취미)
  • She would like to travel to Paris.파리 여행을 하고 싶어해요. (소망)
2
문법 규칙GRAMMAR RULES
LIKE · 긍정
I like coffee.
WOULD LIKE · 요청
I'd like some tea.
LIKE · 부정
I don't like it.
WOULD LIKE · 의문
Would you like...?
  • Like 긍정: I like apples. / She likes music. / They like reading books.
  • Would like 긍정: I would like some tea. / She would like to visit Paris.
  • Like 부정: I don't like cold weather. / She doesn't like spicy food.
  • Would like 부정: I wouldn't like to go alone. / She wouldn't like that dress.
  • 의문문: Do you like pizza? / Would you like some pizza?

긍정문 예시POSITIVE EXAMPLES
  • I like chocolate.초콜릿을 좋아해요.
  • She likes painting.그림 그리기를 좋아해요.
  • We like playing soccer.축구하기를 좋아해요.
  • They like watching movies.영화 보기를 좋아해요.
  • I would like some juice.주스 좀 주세요.

상황별 대화 흐름SITUATIONAL DIALOGUES
🍕 음식 · FOOD
A: Do you like pizza?
B: Yes, I love it!
A: Would you like some coffee?
B: Yes, please.
📚 독서 · READING
A: Does she like reading novels?
B: Yes, especially mysteries.
A: Would he like to join us?
B: I think he would.
🎵 음악 · MUSIC
A: Do you like classical music?
B: Sometimes, when I study.
A: Would you like a window seat?
B: Yes, that would be great.
💼 직장 · WORK
A: Do you like working in teams?
B: Yes, it's more fun.
A: Would you like to go now?
B: Not yet, maybe later.
6
문장 결합 연습SENTENCE COMBINING
I like group projects. (취향) / I would like to join your team. (요청)
→ I don't really like group projects, but I'd like to join your team this time.
She likes traveling by train. (취향) / She would like a window seat. (요청)
→ She likes traveling by train and would like a window seat.
He likes working remotely. (취향) / He would like coffee. (요청)
→ He likes working remotely. Would he like coffee before the meeting?
They don't like spicy food. (취향) / They would like mild options. (요청)
→ They don't like spicy food, so they would like to see the mild options.
She likes cooking with Mom. (취향) / She would like to help with dinner. (요청)
→ She likes cooking with Mom and would like to help with dinner tonight.

8
자주 쓰이는 패턴COMMON PATTERNS
LIKE · 취향 표현
I like + 명사/동명사 — I like coffee. / I like reading.
WOULD LIKE · 요청/바람
I'd like + 명사 / to + 동사원형 — I'd like some tea. / I'd like to go.
축약형 · CONTRACTION
I'd like, You'd like, She'd like, He'd like, We'd like, They'd like
공손한 질문 · POLITE QUESTIONS
Would you like...? — 정중한 요청에 항상 사용하세요.
9
흔한 실수COMMON MISTAKES
✗ I like to eat now. (요청할 때)
✓ I would like to eat now.
현재 순간의 요청에는 would like을 사용합니다.
✗ Do you would like tea?
✓ Would you like tea?
would like의 의문문은 Would를 문장 앞에 놓습니다.
✗ I would like going.
✓ I would like to go.
would like 뒤에는 to + 동사원형을 씁니다.
✗ I like some water. (지금 원할 때)
✓ I would like some water.
지금 원하는 것을 표현할 때는 would like을 사용합니다.
10
레벨업 팁LEVEL-UP TIPS
TIP 1 · 발음
Like: [laɪk] — 단음절, 강하게 발음. Would like: [wʊd laɪk] 또는 축약형 [d laɪk]
TIP 2 · 축약형 활용
회화에서는 I'd like, He'd like, They'd like처럼 축약형을 자주 씁니다.
TIP 3 · 상황 구분
일반 취향을 물을 때는 "Do you like...?", 정중한 제안에는 "Would you like...?"
TIP 4 · 레스토랑에서
주문할 때는 항상 "I'd like..."을 사용하세요. "I want..."보다 훨씬 공손합니다.
11
연습문제PRACTICE EXERCISES

빈칸에 like 또는 would like을 알맞게 넣으세요.

  • I ___ chocolate. (일반 취향) (like)
  • I ___ some water, please. (지금 요청) (would like)
  • She ___ reading novels. (취미) (likes)
  • ___ you ___ to try this? (공손한 제안) (Would ... like)
  • They ___ to order now. (현재 요청) (would like)
