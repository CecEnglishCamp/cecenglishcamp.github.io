Plural Nouns
복수 명사 — 규칙/불규칙 변화

복수 명사 (Plural Nouns): 셀 수 있는 명사가 하나 이상일 때 명사의 형태를 바꾸어 씁니다.

규칙 변화 (Regular): 대부분 명사 끝에 -s 또는 -es를 추가하여 복수형을 만듭니다.

불규칙 변화 (Irregular): 철자나 형태가 완전히 바뀌거나 (e.g., man → men) 단수와 복수의 형태가 같은 (e.g., deer → deer) 예외적인 복수형입니다.

  • book → two books규칙 변화
  • box → three boxes규칙 변화
  • child → four children불규칙 변화
  • man → two men불규칙 변화
2
문법 규칙GRAMMAR RULES
ADD -S
book → books
ADD -ES
box → boxes
IRREGULAR
man → men
NO CHANGE
deer → deer
  • -s 추가 (Most Nouns): book → books, pen → pens
  • -es 추가 (s, sh, ch, x, z로 끝날 때): box → boxes, bus → buses, watch → watches
  • -y → -ies (자음 + y): baby → babies, city → cities (모음 + y는 -s: key → keys)
  • -f/-fe → -ves: leaf → leaves, knife → knives (일부 예외: roof → roofs)
  • 불규칙 변화: man → men, woman → women, child → children, foot → feet, tooth → teeth, mouse → mice

자주 쓰이는 패턴COMMON PATTERNS
규칙 복수형 · REGULAR PLURAL
noun + s / es → books, boxes, buses
I have three books.
-Y → -IES 변환
baby → babies, city → cities
자음 + y로 끝나면 y를 i로 바꾸고 -es를 붙인다.
-F/-FE → -VES 변환
leaf → leaves, knife → knives
f/fe를 v로 바꾸고 -es를 붙인다.
불규칙 복수형 · IRREGULAR
man → men, child → children, foot → feet
형태가 완전히 바뀌는 불규칙 변화.
변화 없음 · NO CHANGE
sheep → sheep, deer → deer, fish → fish
문맥으로 복수를 판단한다.
수량 표현 · QUANTITY
two books, many cities, several knives
수량 형용사와 함께 사용한다.

레벨업 팁LEVEL-UP TIPS
TIP 1 · 발음 주의
'-es'로 끝나는 복수형(boxes, buses, watches)은 /ɪz/ 또는 /əz/로 발음됩니다. (예: box /bɑːks/ → boxes /bɑːksɪz/)
TIP 2 · 불규칙 짝지어 암기
man → men, woman → women, foot → feet, tooth → teeth는 입으로 반복하며 외우는 것이 좋습니다.
TIP 3 · 복수형에 관사 사용
복수 명사 앞에 관사(a/an)는 쓸 수 없지만, 특정 명사를 가리킬 때는 the를 쓸 수 있습니다. (예: I see the books on the table.)
TIP 4 · 집합적 복수
People, police, cattle 등은 형태는 단수처럼 보이나 항상 복수 취급하며 복수 동사를 사용합니다. (예: The people are waiting.)

자가진단 체크리스트SELF-CHECK
  • 기본 -s/-es 규칙을 이해하고 적용할 수 있다.
  • -y → -ies 철자 변화를 기억한다. (e.g., baby → babies)
  • -f/-fe → -ves 철자 변화를 기억한다. (e.g., knife → knives)
  • man, woman, child, foot, tooth의 불규칙 복수형을 말할 수 있다.
  • 복수 명사와 동사의 수일치를 할 수 있다. (e.g., The children are playing)
  • 변화 없는 복수형(sheep, deer, fish)을 안다.
  • 일상 대화에서 복수 명사를 자연스럽게 사용할 수 있다.
