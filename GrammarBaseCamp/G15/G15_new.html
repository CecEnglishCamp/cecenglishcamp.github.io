기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Demonstratives
지시대명사 — this, that, these, those

this (단수/가까움) → 이것, 이 — 가까이 있는 단수 대상을 가리킵니다.

that (단수/멂) → 저것, 저 — 멀리 있는 단수 대상을 가리킵니다.

these (복수/가까움) → 이것들, 이 — 가까이 있는 복수 대상을 가리킵니다.

those (복수/멂) → 저것들, 저 — 멀리 있는 복수 대상을 가리킵니다.

  • This is my pen.가까운 펜 하나
  • That is your bag.먼 가방 하나
  • These are my shoes.가까운 신발 여러 켤레
  • Those are her keys.먼 열쇠 꾸러미
2
문법 규칙GRAMMAR RULES
THIS (가까운 단수)
This is my pen.
this + 단수 명사
THAT (먼 단수)
That is your bag.
that + 단수 명사
THESE (가까운 복수)
These are my shoes.
these + 복수 명사
THOSE (먼 복수)
Those are her keys.
those + 복수 명사
  • this + 단수 명사: this book, this phone (가까운 단수)
  • that + 단수 명사: that chair, that bag (먼 단수)
  • these + 복수 명사: these books, these pens (가까운 복수)
  • those + 복수 명사: those chairs, those shoes (먼 복수)
  • 수량 기준: 단수는 this/that, 복수는 these/those를 사용하며, be동사도 is/are로 일치시켜야 합니다.

실생활 사용 예시REAL-LIFE USAGE
  • This notebook is mine. Those pencils are hers.🏫 학교 — 이 공책은 내 것이고, 저 연필들은 그녀의 것이다.
  • That report is final. These files are for review.💼 직장 — 저 보고서는 최종본이다. 이 파일들은 검토용이다.
  • These shoes are mine. That bag is mom's.🏠 가족 — 이 신발들은 내 것이다. 저 가방은 엄마 것이다.
  • This file is correct. Those links are broken.💻 온라인 — 이 파일이 정확하다. 저 링크들은 깨졌다.
  • That chart is for Room 3. These pills are for today.🏥 병원 — 저 차트는 3호실용이다. 이 약들은 오늘 복용할 것이다.
  • These tickets are ours. That seat is yours.✈️ 여행 — 이 티켓들은 우리의 것이다. 저 좌석은 당신의 것이다.
  • This idea is good. Those suggestions are helpful.📋 회의 — 이 아이디어는 좋다. 저 제안들은 도움이 된다.

레벨업 팁LEVEL-UP TIPS
TIP 1 · 전화 통화
전화 통화 시 자신을 소개할 때는 거리에 상관없이 항상 This를 사용합니다. (예: Hello, This is John speaking.)
TIP 2 · 감정적 거리
That/Those는 물리적인 거리 외에도, 화자가 어떤 대상에 대해 부정적인 감정을 가질 때도 사용될 수 있습니다. (예: I hate that song!)
TIP 3 · 시간적 거리
This/These는 현재 가까운 시간(this week, this morning)을, That/Those는 과거의 먼 시간(that year, those days)을 지칭할 때도 사용됩니다.
TIP 4 · 수일치 강조
지시 형용사가 명사를 수식할 때, 항상 명사의 수와 be동사의 수가 일치하는지 재확인하세요. (This pen is / These pens are)

연습문제PRACTICE EXERCISES

빈칸에 this / that / these / those 중 알맞은 것을 선택하세요.

  • ___ are my friends here. (가까운 복수) (These)
  • Look at ___ star in the sky. (먼 단수) (that)
  • Is ___ pen yours? (가까운 단수) (this)
  • That are my keys. (복수) → Those are my keys.
  • These is my phone. (단수) → This is my phone.
  • 저것들은 네 책이니? → Are those your books?
  • 이 가방은 무거워. → This bag is heavy.
  • 저 신발들은 누구의 것이니? → Whose are those shoes?
