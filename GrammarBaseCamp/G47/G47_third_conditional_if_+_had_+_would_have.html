Third Conditional
과거 가정법 — If + had p.p., would have p.p.
기본 개념 문법 규칙 긍정문 의문문 대화 흐름 문장 결합 실생활 패턴 흔한 실수 레벨업 팁 연습문제 자가진단
1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Third Conditional
과거에 일어나지 않은 일에 대한 상상, 후회, 대안 표현

실제 일어난 일: I didn't study. → I failed the test. / She didn't come. → She couldn't help.

가정 상황: If I had studied, I would have passed. / If she had come, she would have helped.

  • If I had studied, I would have passed.공부했더라면 합격했을 텐데. (후회)
  • If she had come, she would have helped.왔더라면 도와줬을 텐데. (대안)
  • If we had planned better, we would have succeeded.더 잘 계획했더라면 성공했을 텐데. (반성)
2
문법 규칙GRAMMAR RULES
기본 구조 · STRUCTURE
If + had + p.p., would have + p.p.
If I had studied, I would have passed.
순서 변경 · REVERSED
Would have + p.p. + if + had + p.p.
I would have helped if I had known.
COULD HAVE · 가능했던 결과
could have + p.p.
I could have helped you.
MIGHT HAVE · 불확실한 결과
might have + p.p.
I might have succeeded.
  • If + had + p.p, Would have + p.p — 기본 구조
  • "If"절에는 과거완료 사용 (had + 과거분사)
  • "Would have"는 결과절에서 사용
  • 순서 바꾸기 가능: Would have + p.p + if + had + p.p
  • 부정형: If I hadn't missed the bus, I would have arrived on time.
  • 의문형: What would you have done if you had won?
  • "Could have"나 "Might have"로 결과절 표현 가능

자주 쓰이는 패턴COMMON PATTERNS
기본 패턴 · BASIC
If I had known, I would have + p.p.
If I had known, I would have helped.
순서 변경 · REVERSED
I would have + p.p. + if I had + p.p.
I would have helped if I had known.
SHOULD HAVE · 했어야 할 일
I should have + p.p. (후회 표현)
I should have studied more.
Wh-의문문 · WH-QUESTION
What would you have done if + had + p.p.?
What would you have done if you had won?
9
흔한 실수COMMON MISTAKES
✗ If I would have studied, I would have passed.
✓ If I had studied, I would have passed.
If절에는 would have 사용 불가 — had + p.p.만 사용
✗ If I had studied, I would passed.
✓ If I had studied, I would have passed.
결과절에 "have"를 빠뜨리면 안 됨 — would have + p.p.
✗ If I had knew, I would have told you.
✓ If I had known, I would have told you.
had 뒤에는 과거분사(known) 사용, 과거형(knew) 아님
✗ If I studied, I would have passed.
✓ If I had studied, I would have passed.
Third conditional은 If + had + p.p. (단순 과거형 아님)
10
레벨업 팁LEVEL-UP TIPS
TIP 1 · 2nd vs 3rd Conditional 구분
Second: 현재 사실과 다른 상상 (If I had money...) / Third: 과거에 일어나지 않은 일 (If I had studied...)
TIP 2 · should have로 후회 표현
"I should have + p.p."는 "했어야 했는데..."의 의미. 후회할 때 자주 사용됩니다.
TIP 3 · could have / might have 활용
would have = 확실한 결과 / could have = 가능했던 결과 / might have = 불확실한 결과
TIP 4 · 일기 쓰기로 연습
하루를 돌아보며 "If I had..." 문장을 써보세요. 후회/반성/대안을 표현하는 연습이 됩니다.

연습문제PRACTICE EXERCISES

빈칸에 알맞은 형태를 넣으세요.

  • If I ___ (study), I would have passed. (had studied)
  • If she ___ (come), she would have helped. (had come)
  • What ___ you ___ (do) if you had won? (would ... have done)
  • If we ___ (leave) earlier, we would have caught the bus. (had left)
  • I ___ (tell) you if I had known. (would have told)
  • If he ___ (work) harder, he would have succeeded. (had worked)
  • ___ she ___ (help) if she had come? (Would ... have helped)
  • If I ___ (bring) my umbrella, I wouldn't have gotten wet. (had brought)

자가진단 체크리스트SELF-CHECK
  • If + had + p.p., would have + p.p. 구조를 안다.
  • Second Conditional과 Third Conditional의 차이를 안다.
  • should have / could have / might have를 구분할 수 있다.
  • If절에 would have를 쓰면 안 된다는 것을 안다.
  • 후회/대안/반성을 Third Conditional로 표현할 수 있다.
  • 흔한 실수 4가지를 이해했다.
