1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Present Perfect Continuous
과거부터 지금까지 계속되고 있는 동작을 표현

Present Perfect Continuous → 과거부터 지금까지 계속되고 있는 동작을 표현합니다. 주로 지속 시간이나 현재 영향을 강조할 때 사용합니다.

Present Perfect (결과 강조)와 비교: I've worked here for 5 years. (사실) vs I've been working here for 5 years. (지금도 계속 중)

  • I've been studying for 3 hours.3시간째 공부하고 있다. (지속 강조)
  • She has been cooking since 5 PM.5시부터 계속 요리하고 있다. (진행 중)
2
문법 규칙GRAMMAR RULES
긍정문 · POSITIVE
have/has + been + -ing
I have been working.
부정문 · NEGATIVE
have/has + not + been + -ing
I haven't been sleeping.
FOR + 기간 · DURATION
for 2 hours, for a long time
for 3 days, for weeks
SINCE + 시점 · STARTING POINT
since 8 AM, since Monday
since last year, since this morning
  • have/has + been + 동사-ing 구조로 사용
  • 시간 표현: for 2 hours / since 8 AM / all day / recently
  • 현재 영향: I'm tired because I've been studying
  • 진행 강조 시 현재완료진행 / 완료 강조 시 현재완료 사용
  • 기타 표현: lately, recently, all day, all morning, these days

상황별 대화 흐름SITUATIONAL DIALOGUES
🏫 학교 상황 · SCHOOL
A: You look tired.
B: I've been studying all night.
A: What have you been studying?
B: English grammar and writing.
✈️ 여행 상황 · TRAVEL
A: Have you been walking for a long time?
B: Yes, we've been walking since 9 AM.
A: Let's take a break.
B: Good idea.
💼 직장 상황 · OFFICE
A: Where's David?
B: He's been working in the lab all morning.
A: Has he finished the report?
B: Not yet. He's still working.
💻 온라인 상황 · ONLINE
A: Have you been using this app long?
B: I've been using it for a few weeks.
A: Do you like it?
B: Yes, it's very helpful.
🏠 가족 상황 · FAMILY
A: What have you been doing?
B: I've been cleaning the house.
A: Since when?
B: Since this morning.
🏥 병원 상황 · HOSPITAL
A: Have you been feeling sick?
B: Yes, I've been coughing since yesterday.
A: Let's see the doctor.
B: Okay.

→ I'm tired because I've been exercising.
I called him many times today.
→ I've been calling him all day.

→ She has been living here since 2020.

→ He has been studying since 6 PM.
They started playing soccer this morning.
→ They've been playing soccer since morning.

→ She has been practicing piano for two hours.

자주 쓰이는 패턴COMMON PATTERNS
지속 동작 · ONGOING ACTION
I've been + -ing + for/since + 시간
I've been reading this book for a week.
현재 결과 · PRESENT RESULT
I'm tired because I've been + -ing
I'm tired because I've been exercising.
반복 행동 · REPEATED ACTION
I've been + -ing + all day/recently
I've been calling him all day.
How long 질문 · DURATION QUESTION
How long have you been + -ing?
How long have you been studying?
9
흔한 실수COMMON MISTAKES
✗ I have been study for 3 hours.
✓ I have been studying for 3 hours.
been 뒤에는 반드시 동사-ing 형태 사용
✗ She has been cooking for 5 PM.
✓ She has been cooking since 5 PM.
시점(5 PM)은 since, 기간(2 hours)은 for 사용
✗ I have been knowing him for years.
✓ I have known him for years.
know, love, want 등 상태 동사는 진행형 불가
✗ He been working all day.
✓ He has been working all day.
주어 뒤에 have/has를 빠뜨리면 안 됨
10
레벨업 팁LEVEL-UP TIPS
TIP 1 · for vs since 구분
for + 기간 (for 3 hours, for a week) / since + 시점 (since 8 AM, since Monday)
TIP 2 · 상태 동사 주의
know, love, want, believe, own 등 상태 동사는 현재완료진행 사용 불가! 현재완료만 사용하세요.
TIP 3 · 현재완료 vs 현재완료진행
결과/완료 강조 → 현재완료: I've read 3 books. / 과정/지속 강조 → 현재완료진행: I've been reading all day.
TIP 4 · 실생활 연습
"요즘 뭐 하고 있어?" → "I've been studying / I've been working on..." 패턴으로 연습하세요.

자가진단 체크리스트SELF-CHECK
  • have/has been + -ing 구조를 안다.
  • for와 since를 올바르게 구분할 수 있다.
  • 상태 동사는 진행형으로 쓸 수 없다는 것을 안다.
  • 현재완료와 현재완료진행의 차이를 설명할 수 있다.
  • How long have you been...? 질문을 만들 수 있다.
  • 일상 대화에서 자연스럽게 사용할 수 있다.
