1. 기본 개념 (Basic Concept)

PASSPassive Voice

수동태(Passive Voice): 주어가 동작을 받는 문장 구조
능동태: 주어가 행동함 → 수동태: 주어가 행동을 당함
기본 구조: be + 과거분사(p.p.)
행위자보다 결과나 상태에 초점
be + 과거분사 (Past Participle)

능동: They cleaned the room.

수동: The room was cleaned.

능동: She will write a report.

수동: A report will be written.

2. 문법 규칙 (Grammar Rules)

  • 현재: is/are + p.p. → The book is read.
  • 과거: was/were + p.p. → The book was read.
  • 현재완료: has/have been + p.p. → The book has been read.
  • 과거완료: had been + p.p. → The book had been read.
  • 미래: will be + p.p. → The book will be read.
  • 진행형: is/was being + p.p. → The book is being read.
  • 조동사: can/must/should be + p.p. → The book must be read.
  • 행위자(by + 사람)는 생략 가능
  • 시제에 따라 be동사 형태가 달라짐

5. 상황별 대화 흐름 (Situational Dialogues)

학교 상황
A: Did you submit the homework?
B: Yes, it was submitted this morning.
A: Great! I'm sure it'll be graded soon.
여행 상황
A: Is the room ready for us?
B: It is being cleaned right now.
A: Okay, we'll wait in the lobby.
직장 상황
A: Has the report been sent to the client?
B: Yes, it has been emailed to them.
A: Perfect. They should receive it shortly.
온라인 상황
A: Will the results be posted today?
B: Yes, they will be uploaded tonight.
A: I'll check the website later then.
가족 상황
A: Who cleaned the kitchen so well?
B: It was cleaned by Dad.
A: He always does such a thorough job.
병원 상황
A: Should the medicine be taken now?
B: Yes, it should be taken before lunch.
A: Got it, I'll take it right away.
쇼핑 상황
A: Can this item be returned?
B: Yes, it can be returned within 30 days.
A: That's good to know, thanks.
회의 상황
A: When was this decision made?
B: It was decided in yesterday's meeting.
A: I see, I missed that discussion.

  • 현재: is/are + p.p. → The book is read.
  • 과거: was/were + p.p. → The book was read.
  • 현재완료: has/have been + p.p. → The book has been read.
  • 과거완료: had been + p.p. → The book had been read.
  • 미래: will be + p.p. → The book will be read.
  • 진행형: is/was being + p.p. → The book is being read.
  • 조동사: can/must/should be + p.p. → The book must be read.
  • 의문문: Is the report finished?
  • 부정문: The task wasn't completed on time.
  • 행위자 표시: The letter was written by John.
  • 현재진행 수동: The problem is being solved.

  • 미래완료 수동: The work will have been done.
  • 조건문 수동: If it is completed, we can proceed.
  • 명령문 수동: Let the door be opened.

9. 흔한 실수 (Common Mistakes)

❌ The room cleaned → ✅ The room was cleaned
❌ The report is write → ✅ The report is written
❌ The homework has been check → ✅ The homework has been checked
❌ The email will sent → ✅ The email will be sent
❌ The task should complete → ✅ The task should be completed
❌ The book is reading → ✅ The book is being read
❌ The work will finished → ✅ The work will be finished
❌ The letter was wrote → ✅ The letter was written

10. 레벨업 팁 (Level-Up Tips)

  • 수동태는 be + p.p. 구조가 핵심
  • 시제에 따라 be동사 형태가 달라짐
  • 행위자(by + 사람)는 생략 가능, 강조할 때만 사용
  • 진행형 수동태는 is/was being + p.p.
  • 완료형 수동태는 has/have/had been + p.p.
  • 조동사 수동태는 can/must/should be + p.p.
  • 문장 결합 연습을 통해 시제별 수동태에 익숙해지기
  • 과거분사 형태 정확히 기억하기 (regular vs irregular verbs)
  • 수동태는 객관적이고 정중한 표현에 유용
  • 공식 문서나 학술 글에서 자주 사용됨
  • 행위자를 모르거나 결과에 집중할 때 효과적
  • 비즈니스 영어에서 책임을 돌리지 않으려 할 때 사용
  • 뉴스나 보고서에서 객관성을 유지하려 할 때 활용
  • 수동태로 문장의 주어를 바꿔 강조점 변경 가능
  • 과정이나 절차를 설명할 때 수동태가 자연스러움

11. 연습문제 (Practice Exercises)

객관식 문제

1. Which is correct?
A. The report was write.
B. The report was written.
정답: B. "The report was written." - 수동태에서는 과거분사(written)를 사용해야 합니다.
2. Choose the correct form: The homework ___ every day.
A. is checked
B. checks
정답: A. "is checked" - 수동태 구조 "be + 과거분사"를 사용합니다.
3. Fill in: The email ___ before noon.
A. had been sent
B. has sent
정답: A. "had been sent" - 과거완료 수동태로 "had been + 과거분사"를 사용합니다.
4. Which sentence shows correct passive voice?
A. The room is cleaning now.
B. The room is being cleaned now.
정답: B. "The room is being cleaned now." - 현재진행 수동태는 "is/are being + 과거분사"입니다.
5. Complete the sentence: The task ___ by tomorrow.
A. will complete
B. will be completed
정답: B. "will be completed" - 미래 수동태는 "will be + 과거분사"입니다.
