1. 기본 개념 (Basic Concept)

3CTHIRD CONDITIONAL

과거에 일어나지 않은 일에 대해 상상, 후회, 대안을 표현
실제로는 일어나지 않았지만 그랬다면 결과가 달랐을 것을 말함

후회: If I had studied, I would have passed.

대안: If she had come, she would have helped.

반성: If we had planned better, we would have succeeded.

기본 구조

If + had + 과거분사, Would have + 과거분사

예시: If I had studied, I would have passed.

실제 vs 가정 상황

실제 일어난 일 (Reality)

I didn't study.
→ I failed the test.
(공부하지 않았다 → 시험에 실패했다)

She didn't come.
→ She couldn't help.
(오지 않았다 → 도울 수 없었다)

가정 상황 (What could have been)

If I had studied,
→ I would have passed.
(공부했더라면 → 합격했을 텐데)

If she had come,
→ She would have helped.
(왔더라면 → 도와줬을 텐데)

시간 흐름 이해하기

과거 조건 (Didn't happen)

If I had studied
(실제로는 공부 안 함)

→
가상 결과 (Would have been)

I would have passed
(합격했을 것임)

표현 목적별 사용법

후회 표현
If I had saved money...
대안 상상
If we had taken the train...
조언/비판
If you had listened...
감정 표현
If she had called...

2. 문법 규칙 (Grammar Rules)

기본 구조와 규칙

  • If + had + p.p, Would have + p.p
  • "If"절에는 과거완료 사용 (had + 과거분사)
  • "Would have"는 결과절에서 사용
  • 순서 바꾸기 가능: Would have + p.p + if + had + p.p
  • 부정형: If I hadn't missed the bus, I would have arrived on time.
  • 의문형: What would you have done if you had won?
  • "Could have"나 "Might have"로 결과절 표현 가능

다양한 결과 표현

  • Would have: 확실한 결과 → I would have called you.
  • Could have: 가능했던 결과 → I could have helped you.
  • Might have: 불확실한 결과 → I might have succeeded.
  • Should have: 했어야 할 일 → I should have studied more.

문장 순서와 변형

  • 기본: If I had known, I would have helped.
  • 순서 변경: I would have helped if I had known.
  • 의문문: Would you have helped if you had known?
  • Wh-의문문: What would you have done if you had known?

3. 예시 문장 (Example Sentences)

긍정문 (Positive Examples)

  • If I had studied, I would have passed the test.
  • If she had come, she would have helped us.
  • If we had left earlier, we would have caught the bus.
  • If he had listened, he would have understood.
  • If I had known, I would have told you.
  • If they had invited me, I would have gone.
  • If I had brought my umbrella, I wouldn't have gotten wet.
  • If we had booked the tickets, we would have gone to the concert.
  • If he had worked harder, he would have succeeded.
  • If I had remembered, I would have called you.
