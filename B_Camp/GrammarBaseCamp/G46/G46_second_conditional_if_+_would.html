1. 기본 개념 (Basic Concept)

2CSECOND CONDITIONAL

현재 사실과 다른 상상이나 희망/조언을 표현
실제로 일어나지 않았지만 가능할 수도 있는 상황을 말함

상상: If I had wings, I would fly.

희망: If I had more time, I would study more.

조언: If I were you, I would talk to him.

기본 구조

If + 과거, Would + 동사원형

예시: If I had money, I would travel.

현실 vs 가정 비교

현실 (Reality)

I don't have much money.
(나는 돈이 많지 않다.)

She isn't here.
(그녀는 여기 없다.)

가정 (If-situation)

If I had money, I would travel.
(돈이 있다면, 여행을 할 텐데.)

If she were here, she would help.
(그녀가 여기 있다면, 도와줄 텐데.)

사용 목적별 분류

상상 표현
If I lived in space...
희망 표현
If I had more time...
조언 표현
If I were you...
예의 표현
Would you help me if...

2. 문법 규칙 (Grammar Rules)

기본 구조와 규칙

  • If + 과거, Would + 동사원형
  • "If"절에는 과거형 사용 (be동사는 "were" 권장)
  • "Would"는 결과절에서 사용
  • 순서 바꾸기 가능: Would + 주어 + 동사원형 + if + 과거
  • 부정형: If I didn't have homework, I would go out.
  • 의문형: What would you do if you won the lottery?
  • "Could"나 "Might"로 결과절 표현 가능: I might go / I could help

Be 동사 특별 규칙

  • 모든 주어에 대해 "were" 사용 (formal)
  • If I were rich... / If he were here... / If she were taller...
  • "Was"도 사용 가능하지만 "were"가 더 정확함
  • "If I were you..."는 조언할 때 자주 사용하는 표현

문장 순서 변경

  • 기본: If I had money, I would travel.
  • 순서 변경: I would travel if I had money.
  • 의문문: Would you travel if you had money?
  • Wh-의문문: What would you do if you had money?

3. 예시 문장 (Example Sentences)

긍정문 (Positive Examples)

  • If I had more money, I would travel the world.
  • If she were here, she would help us.
  • If we lived near the beach, we would swim every day.
  • If he studied harder, he would pass the test.
  • If I knew the answer, I would tell you.
  • If they invited me, I would go.
  • If I were taller, I would play basketball.
  • If we had a car, we would drive to the mountains.
  • If it didn't rain, we would play outside.
  • If I had more time, I would read more books.

의문문 (Question Examples)

  • What would you do if you won the lottery?
  • If you had a free day, what would you do?
  • Would you help me if I asked?
  • If you lived in another country, where would you live?
  • Would you go to the party if they invited you?
  • If you were rich, what would you buy?
  • Would you study abroad if you had the chance?
  • If you didn't have homework, what would you do?
  • Would you travel more if you had time?
  • If you could change one thing, what would it be?

4. 상황별 대화 (Situational Dialogues)

학교 상황 (School Situation)
A: If I had more time, I would study grammar.
B: Same here. I would read more too.
A: Would you join the club if you didn't have homework?
B: Yes, I would!
여행 상황 (Travel Situation)
A: If we had a car, we would visit more places.
B: True. We would go to the mountains.
A: Would you travel alone if you had the chance?
B: Maybe. I would try it once.
직장 상황 (Work Situation)
A: If I worked from home, I would save time.
B: I would be more relaxed too.
A: Would you move to another city if you got a better job?
B: I think I would.
