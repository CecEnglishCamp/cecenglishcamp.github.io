1. 기본 개념 (Basic Concept)

PPCPRESENT PERFECT CONTINUOUS

과거부터 지금까지 계속되고 있는 동작을 표현
주로 지속 시간이나 현재 영향을 강조할 때 사용

지속 강조: I've been studying for 3 hours.

현재 영향: I'm tired because I've been working.

진행 중: She has been cooking since 5 PM.

기본 구조

have/has + been + 동사-ing

예시: I have been working / She has been studying

Present Perfect vs Present Perfect Continuous

Present Perfect (결과 강조):
I've worked here for 5 years. (5년간 일한 경험/사실)
Present Perfect Continuous (지속 강조):
I've been working here for 5 years. (5년째 계속 일하고 있음)

2. 문법 규칙 (Grammar Rules)

문법 구조

  • have/has + been + 동사-ing
  • 시간 표현: for 2 hours / since 8 AM / all day / recently
  • 현재 영향: I'm tired because I've been studying
  • 진행 강조 시 현재완료진행 / 완료 강조 시 현재완료 사용

시간 표현

FOR + 기간

  • for 2 hours
  • for a long time
  • for 3 days
  • for weeks

SINCE + 시점

  • since 8 AM
  • since Monday
  • since last year
  • since this morning

기타 표현

  • lately, recently
  • all day
  • all morning
  • these days

사용 목적

  • 지속된 동작: I've been reading this book for a week.
  • 현재 결과: I'm tired because I've been exercising.
  • 반복 행동: I've been calling him all day.
  • 진행 중인 상황: She has been living here since 2020.

3. 예시 문장 (Example Sentences)

긍정문 (Positive Examples)

  • I've been studying for 3 hours.
  • She has been cooking since 5 PM.
  • We've been waiting all day.
  • He has been working here for 10 years.
  • I've been thinking about that problem.
  • They've been playing soccer since morning.
  • She has been practicing piano for two hours.
  • I've been feeling sick lately.
  • We've been talking about this issue.
  • He has been fixing the car since noon.

의문문 (Question Examples)

  • How long have you been studying?
  • Have you been working all day?
  • What have you been doing?
  • Has she been cooking since morning?
  • Have they been playing soccer?
  • Have you been feeling okay?
  • Has he been working here long?
  • Have you been waiting for me?
  • What have you been thinking about?
  • Have we been talking too much?

4. 상황별 대화 (Situational Dialogues)

학교 상황 (School Situation)
A: You look tired.
B: I've been studying all night.
A: What have you been studying?
B: English grammar and writing.
여행 상황 (Travel Situation)
A: Have you been walking for a long time?
B: Yes, we've been walking since 9 AM.
A: Let's take a break.
B: Good idea.
직장 상황 (Work Situation)
A: Where's David?
B: He's been working in the lab all morning.
A: Has he finished the report?
B: Not yet. He's still working.
온라인 상황 (Online Situation)
A: Have you been using this app long?
B: I've been using it for a few weeks.
A: Do you like it?
B: Yes, it's very helpful.
가족 상황 (Family Situation)
A: What have you been doing?
B: I've been cleaning the house.
A: Since when?
B: Since this morning.
병원 상황 (Hospital Situation)
A: Have you been feeling sick?
B: Yes, I've been coughing since yesterday.
A: Let's see the doctor.
B: Okay.
