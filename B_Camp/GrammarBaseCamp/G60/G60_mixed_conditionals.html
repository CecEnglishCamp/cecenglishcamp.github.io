1. 기본 개념 (Basic Concept)

MIXEDMIXED CONDITIONALS

조건절과 결과절의 시제가 서로 다른 조건문
실제 상황과 다른 가정을 표현함
복합적인 시간 관계를 정확하게 표현할 수 있음

과거→현재: If I had studied, I would know the answer.

현재→과거: If I were taller, I would have reached the shelf.

과거 조건 → 현재 결과
과거의 행동/결정이 현재에 미치는 영향
If I had slept well, I would feel better now.
현재 조건 → 과거 결과
현재의 능력/상황이 과거에 미쳤을 영향
If I were taller, I would have reached it.

구조 비교 가이드

Type 1: If + past perfect, would + base verb (과거→현재)
Type 2: If + past simple, would have + past participle (현재→과거)

2. 문법 규칙 (Grammar Rules)

과거 조건 → 현재 결과

  • 구조: If + past perfect, would + base verb
  • 예시: If I had studied, I would know the answer.
  • 의미: 과거에 하지 않은 일이 현재에 미치는 영향
  • 감정: 후회, 아쉬움 등을 표현

현재 조건 → 과거 결과

  • 구조: If + past simple, would have + past participle
  • 예시: If I were taller, I would have reached the shelf.
  • 의미: 현재의 능력/상황 부족이 과거에 미쳤을 영향
  • 감정: 능력 한계, 상황적 제약 등을 표현

주요 규칙

  • 혼합 조건문은 시제 일치가 중요
  • 주절과 조건절의 시간 관계를 정확히 파악해야 함
  • 조동사: would, could, might 사용 가능
  • 부정형: wouldn't, couldn't, might not

3. 예시 문장 (Example Sentences)

과거 조건 → 현재 결과

  • If I had slept well, I would feel better now.
  • If she had studied, she would know the answer.
  • If they had arrived earlier, they would be here now.
  • If we had taken the other road, we would be at the hotel.
  • If he had eaten breakfast, he wouldn't be hungry now.
  • If I had listened to you, I wouldn't be in trouble.
  • If she had saved money, she could buy it now.
  • If we had left earlier, we would be there by now.

현재 조건 → 과거 결과

  • If I were taller, I would have reached the shelf.
  • If she were here, she would have helped us.
  • If we had more money, we would have bought it.
  • If he were stronger, he would have lifted it.
  • If they spoke English, they would have understood the joke.
  • If I were more confident, I would have applied for the job.
  • If she were available, she could have joined us.
  • If we were organized, we wouldn't have missed the deadline.

8. 표현 패턴 (Expression Patterns)

후회 표현
If I had + past participle, I would + base verb + now

• If I had listened to you, I would be safe now.

• If I had taken the medicine, I would feel better now.

능력 부족 표현
If I were + adjective, I would have + past participle

• If I were braver, I would have spoken up.

• If I were stronger, I would have helped you.

상황 가정
If + situation were different, would have + result

• If the weather were better, we would have gone out.

• If the store were open, I would have bought it.

과거 결정의 현재 영향
If I had + decision, I would + current state

• If I had moved to Seoul, I would live there now.

• If I had learned French, I would speak it now.

현재 상태의 과거 영향
If I were + state, I would have + past action

• If I were more organized, I would have finished on time.

• If I were taller, I would have reached it easily.

조건과 결과의 시간차
Time-shifted cause and effect

• If we had left early, we would be relaxing now.

• If I were rich, I would have bought that house.

9. 흔한 실수 (Common Mistakes)

실수 교정 가이드

❌ If I would have studied, I would know it now.
✅ If I had studied, I would know it now.
조건절에서는 would를 사용하지 않습니다.
❌ If I was taller, I would have reached it.
✅ If I were taller, I would have reached it.
가정법에서는 were를 사용합니다.
❌ If I had studied, I will know it now.
✅ If I had studied, I would know it now.
주절에서는 would를 사용해야 합니다.
❌ If I am taller, I would have reached it.
✅ If I were taller, I would have reached it.
현재 가정 상황은 과거형을 사용합니다.
❌ If I had ate breakfast, I wouldn't be hungry.
✅ If I had eaten breakfast, I wouldn't be hungry.
과거완료는 had + 과거분사입니다.
❌ If she were here, she will have helped us.
✅ If she were here, she would have helped us.
가정법에서는 would have를 사용합니다.
❌ If we left earlier, we would be there now.
✅ If we had left earlier, we would be there now.
과거 행동의 현재 결과는 과거완료를 사용합니다.
❌ If I have more money, I would have bought it.
✅ If I had more money, I would have bought it.
현재 상황의 과거 결과는 과거형을 사용합니다.
