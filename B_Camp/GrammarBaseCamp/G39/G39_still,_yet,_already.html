1. 기본 개념 (Basic Concept)

STSTILL

어떤 일이 계속되고 있음을 나타냄
(현재진행/현재시제)

현재진행: I'm still working.

현재시제: She still lives here.

상태: I still love you.

YTYET

어떤 일이 아직 발생하지 않음 또는 이미 발생했는지 확인
(현재완료/의문문/부정문)

부정문: I haven't eaten yet.

의문문: Have you finished yet?

확인: Did they arrive yet?

ALALREADY

어떤 일이 이미 발생함을 나타냄
(현재완료/과거시제)

현재완료: I've already finished.

과거시제: I already told you.

놀람: She already left?

비교 예시

Are you still working? (계속 일하는 중인지 확인)
Have you finished yet? (끝났는지 확인)
I've already finished. (이미 끝났음)

2. 문법 규칙 (Grammar Rules)

기본 문법 규칙

  • Still: 일반적으로 긍정문에서 사용, 동사 앞에 위치 (She still lives here)
  • Yet: 의문문/부정문에서 사용, 문장 끝에 위치 (Have you eaten yet?)
  • Already: 긍정문에서 사용, 현재완료 시제와 자주 결합 (I've already eaten)
  • Still은 현재진행/현재시제와 자주 사용됨
  • Yet은 현재완료/미완료 동작 확인에 사용됨
  • Already는 놀람/예상보다 빠른 완료를 표현함

시제별 사용법

  • Still + 현재진행형 → I'm still waiting / She's still studying
  • Still + 현재시제 → I still live here / He still works there
  • Yet + 현재완료 → I haven't finished yet / Has she called yet?
  • Already + 현재완료 → I've already done it / They've already left
  • Already + 과거시제 → I already saw that / She already paid

3. 예시 문장 (Example Sentences)

Still Examples

  • I still live in Seoul.
  • She still works at that company.
  • We still have time.
  • He still loves that song.
  • They still play soccer every weekend.
  • I'm still thinking about it.
  • She's still studying.
  • He still hasn't called. (강조용 부정)
  • We're still waiting.
  • It's still raining.

Already Examples

  • I've already finished my homework.
  • She's already gone home.
  • We've already seen that movie.
  • He already knows the answer.
  • They've already eaten.
  • I already told you.
  • She already bought the tickets.
  • We already started the meeting.
  • He's already sleeping.
  • I've already packed my bag.

의문문 예시 (Question Examples)

  • Are you still working?
  • Is she still here?
  • Do you still want to go?
  • Have you finished yet?
  • Has he called yet?
  • Did they arrive yet?
  • Have you eaten yet?
  • Have you already seen this?
  • Did she already leave?
  • Have they already paid?

4. 상황별 대화 (Situational Dialogues)

학교 상황 (School Situation)
A: Have you finished your homework yet?
B: No, I'm still working on it.
A: I already finished mine.
B: Lucky you!
여행 상황 (Travel Situation)
A: Did you book the hotel yet?
B: Yes, I already did it.
A: Are you still planning the schedule?
B: Yes, I need more time.
직장 상황 (Work Situation)
A: Have they submitted the report yet?
B: No, they're still editing.
A: I already sent mine.
B: You're always early!
온라인 상황 (Online Situation)
A: Have you logged in yet?
B: Yes, I already joined the meeting.
A: Is Sarah still offline?
B: Yes, she hasn't joined yet.
가족 상황 (Family Situation)
A: Did Mom cook dinner yet?
B: No, she's still preparing.
A: I already set the table.
B: Great, thanks!
병원 상황 (Hospital Situation)
A: Has the doctor arrived yet?
B: No, we're still waiting.
A: I already filled out the form.
B: That helps a lot.

5. 문장 결합 연습 (Sentence Combining Practice)

문장 변환 연습

I'm working. It's not finished.
→ I'm still working.
