우주비행사는 산소 시스템을 어떻게 보고할까?

Space Camp·2026. 6. 15.·3 min read

우주 영화를 보면 비행사가 이렇게 말하는 장면이 자주 나옵니다.

👨‍🚀 Astronaut:

"Oxygen system check. Tank pressure is nominal. Regulator is stable. Over."

🎧 Mission Control:

"Copy. Oxygen system is nominal. Proceed."

산소 시스템(Oxygen System)은 우주에서 가장 중요한 장비입니다. 지구 밖에서는 단 15초만 산소가 끊겨도 의식을 잃을 수 있기 때문에, 우주비행사는 산소 시스템 상태를 자주 그리고 정확하게 보고해야 합니다.

이 글에서는 산소 시스템 점검과 보고에 쓰이는 핵심 영어 표현을 정리합니다.

산소 시스템 핵심 단어

영어뜻예문
oxygen system산소 시스템"Oxygen system is fully operational."
oxygen tank산소 탱크"O2 tank pressure at 3200 PSI."
pressure압력"Pressure is within limits."
regulator조절기"Regulator is functioning normally."
oxygen flow산소 흐름"Oxygen flow rate is stable."
primary / backup주 / 예비"Switching to backup oxygen supply."
EVA (Extravehicular Activity)우주유영"Pre-breathing for EVA complete."

산소 시스템 보고 대화 예시

👨‍🚀 Astronaut:

"Houston, this is Artemis 2. Initiating oxygen system check. Over."

🎧 Mission Control:

"Go ahead, Artemis 2. Over."

👨‍🚀 Astronaut:

"Primary tank pressure: 3200 PSI. Backup tank: 3200 PSI. Regulator: stable. Oxygen flow: 0.5 LPM. All nominal. Over."

🎧 Mission Control:

"Copy all. Oxygen system nominal. You are go for EVA. Over."

핵심 표현: "All nominal" 다시 보기

앞선 블로그 글에서 "All nominal"의 뜻을 배웠습니다. 산소 시스템 보고에서 "All nominal"은 "모든 수치가 계획된 정상 범위 내에 있다"는 뜻입니다. NASA에서는 이 짧은 두 단어로 탱크 압력, 유량, 조절기 상태를 한 번에 보고합니다.

초등학생이 가정에서 연습할 때는 이렇게 간단히 말해보세요:

👨‍🚀 (아이):

"Oxygen system is working. All nominal. Over."

🎧 (부모):

"Copy. Good job. Over."

우주영어 루틴: 하루 한 문장

아이와 함께 아래 문장을 하루에 하나씩 연습해보세요. 7일이면 산소 시스템 보고를 영어로 할 수 있습니다.

  1. Day 1: "Oxygen system is working."
  2. Day 2: "Tank pressure is nominal."
  3. Day 3: "Regulator is stable."
  4. Day 4: "Oxygen flow is normal."
  5. Day 5: "All systems nominal."
  6. Day 6: "Switching to backup."
  7. Day 7: "Oxygen system check complete. All nominal. Over."
